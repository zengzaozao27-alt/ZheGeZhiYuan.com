 浙个志愿

> 浙江高考志愿填报参考工具 · 基于 2023-2025 年真实投档数据（数据已修正）

---

 简介

这是一个面向浙江高考生的志愿填报参考工具，纯前端单文件实现，无需服务器、无需安装，浏览器打开即可使用。

数据来源为浙江省教育考试院公开发布的《浙江省普通高校招生投档及专业录取情况》（2023-2025 年），覆盖普通类平行投档与艺术类统考批两大板块。v4 版本已基于官方 docx 原始文件重新提取并修正了全部数据，解决了此前 PDF 解析中部分学校专业数据异常的问题。

声明：本工具与浙江省教育考试院及任何高校官方无关，仅基于公开数据整理，供考生和家长参考决策。**

---

 核心功能

| 功能 | 说明 |
|------|------|
| 双模式报考 | 支持「普通类」与「艺术统考类」切换，艺术模式仅推荐省统考批专业，不涉及校考 |
| 按学校分组 | 结果以学校为卡片聚合，一眼看到该校有哪些专业可报（保底 / 稳妥 / 冲刺） |
| 档位自动计算 | 根据用户位次与历年录取位次的比值，自动标记专业为「冲刺」「稳妥」「保底」 |
| 历年数据对照 | 展开专业列表后，可切换查看 2023 / 2024 / 2025 三年分数与排名变化 |
| 选科过滤 | 自动过滤不符合用户选考科目要求的专业，防止退档风险 |
| **多维筛选** | 顶部直接展示筛选栏（地区、省份、城市、学校层次），即时联动 |

- 普通类：2023-2025 年在浙江省招生的普通高校平行投档数据，约 1,874 所高校、51,000+ 专业志愿记录，已修复学校/专业名称括号格式差异导致的合并错误

 使用提示

1. 位次比分数更重要：输入时请优先确保全省排名（位次）准确，这是档位计算的核心依据。
2. 艺术生注意：艺术统考模式使用的是**综合分位次**，不是纯文化分位次。综合分 = 高考总分 × 50% + 省统考成绩 × 2.5 × 50%。
3. 选科过滤：系统会自动排除你选考科目不满足要求的专业，但请务必最终核对高校招生章程中的具体要求。
4. 大小年波动：历年数据仅供参考，热门学校/专业可能存在录取位次上下波动，建议冲稳保结合填报。
5. 务必服从调剂：平行志愿模式下，除非有十足把握，否则建议勾选服从调剂，防止退档。

 已知局限

- 学校层次标签基于公开资料映射，新升格院校、更名院校可能存在标注滞后。
- 艺术统考数据中的位次为各艺术科类独立排名（B=美术、C=音乐、D=舞蹈、E=表导演、G=播音、M=书法），不可与普通类位次混用。
- 未包含军校、公安、三位一体、强基计划等特殊招生类型的数据。
